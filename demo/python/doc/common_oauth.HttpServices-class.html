        HttpServices

HTTP data and service functions

This class defines URLs and associated data needed to interact with Dropbox. It includes methods to manage the HTTPD session and to open a web browser for the user. NOTE THAT THIS DATA MUST MATCH THE INFORMATION ENTERED AT THE dROPBOX DEVELOPER CONSOLE FOR THE APP
