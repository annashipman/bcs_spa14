oauth_client.py: Dropbox Python OAuth API demo client

This is the script which runs the Oauth demo client. It checks whether the access token file exists, and presents the user with a menu of relevant options.

If the client has previously been authorised with Dropbox (ie the Dropbox token file exists), the options allow the user to manipulate files in the Dropbox app directory using Dropbox calls (not using filesystem I/O).

If the client has not yet been authorised with Dropbox (ie the Dropbox token file does not exist), the options allow the user to run either the no-redirect or redirect workflows.
